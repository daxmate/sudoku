# 🧩 数独游戏

一个纯前端、开箱即玩的数独游戏，支持四种难度、笔记/候选编辑、提示推理、键盘操作。

## ✨ 特性

- **四种难度** — 简单 / 中等 / 困难 / 专家（空格数 32–54）
- **全部标记** 💣 — 一键为所有空格计算候选数字
- **编辑标记** ✏️ — 点击铅笔按钮进入候选编辑模式，手动增减候选
- **单格候选** 🔢 — 只给当前选中格计算候选数
- **提示系统** 💡 — 唯余法 → 隐唯法 → 揭晓答案，逐步推理
- **错误计数** — 最多 3 次错误，超限游戏结束
- **计时器** — 自动计时，记录解题用时
- **键盘支持** — 数字键填入，方向键移动，Backspace 擦除
- **智能高亮** — 选中格高亮同行/同列/同宫，相同数字蓝色高亮
- **数字耗尽** — 某数字填满 9 个后自动禁用对应按键
- **响应式** — 桌面双栏布局，移动端窄屏自动堆叠
- **缩放滑块** — 可调节界面大小（70%–130%），自动保存
- **素雅配色** — 低饱和度色彩，温和护眼

## 🚀 使用

直接打开 `sudoku.html` 即可游玩 —— 无需安装、无需服务器。

```bash
open sudoku.html
```

或部署到任何静态托管服务（GitHub Pages、Netlify 等）。

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `1`–`9` | 填入数字 |
| `Backspace` / `Delete` | 擦除当前格数字 |
| `↑ ↓ ← →` | 移动选中格 |


## 🧠 技术细节

- **核心引擎** (`SudokuEngine`) — 回溯算法生成唯一解谜题，完整求解器，候选计算
- **UI 层** — 原生 DOM 操作，零依赖
- **布局** — 双栏布局（棋盘左 + 操作面板右）
- **图标** — 内联 SVG 图标，跟随 `currentColor` 配色
- **弹窗** — 自定义浮层替代浏览器原生 `alert` / `confirm`
- **缩放** — 使用 CSS `zoom` 属性，滑块 + localStorage 持久化
- **风格** — 毛玻璃设计 + 渐变背景，低饱和素雅配色
