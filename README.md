# 🧩 数独游戏

一个纯前端、开箱即玩的数独游戏，支持四种难度、笔记模式、键盘操作。

## ✨ 特性

- **四种难度** — 简单 / 中等 / 困难 / 专家（空格数 32–54）
- **一键标记** — 自动为所有空格填入行列宫中缺失的候选数字
- **智能高亮** — 选中格时高亮同行、同列、同宫，相同数字蓝色高亮
- **错误计数** — 最多 3 次错误，超限游戏结束
- **计时器** — 自动计时，记录你的解题用时
- **提示系统** — 随机揭示一个正确答案
- **一键解答** — 自动填入所有答案
- **键盘支持** — 数字键填入，方向键移动，N 切换笔记模式
- **响应式** — 桌面和移动端均可使用

## 🚀 使用

直接打开 `sudoku.html` 即可游玩 —— 无需安装、无需服务器。

```bash
open sudoku.html
```

或部署到任何静态托管服务（GitHub Pages、Netlify 等）。

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `1`–`9` | 填入数字 |
| `Backspace` / `Delete` | 擦除当前格 |
| `一键标记` | 自动标记所有空格的候选数字 |
| `↑ ↓ ← →` | 移动选中格 |

## 🧠 技术细节

- **核心引擎** (`SudokuEngine`) — 回溯算法生成唯一解谜题，完整求解器
- **UI 层** — 原生 DOM 操作，零依赖
- **风格** — 毛玻璃设计 + 渐变背景
